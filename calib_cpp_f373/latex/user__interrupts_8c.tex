Trigger button input state change handler. Signal the appropriate rising or falling edge handler according to G\+P\+IO state and debounce routine. 

If the button has been pressed and the debounce flag is not active, set the debounce timer, raise the flag, and call the handler. Pass if debounce flag is active. ~\newline
 If the button has been released and the debounce flag is not active, set the debounce timer, raise the flag, and call the handler. Pass if debounce flag is active.
